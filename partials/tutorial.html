Markdown is a pretty easy yet powerful plain text markup language that can be used to format text in various ways but without many complicated tags. In fact it can be read and written very easily. This editor parses the entered Markdown and displays the result as HTML instantly. To write Markdown, you can use the following tags:

A heading is written as a normal text but with multiple = characters as below: 

Heading
=======

Sub-Headers work the same way, just use multiple - characters below the actual sub-heading:

Sub-Heading
-----------

Text can be bold, italic and monospaced like this:

**bold** *italic* `monospace`

Lists can be easily formatted with an asterisk. For ordered lists just use numbers:

* cereals
* cocoa
* milk

1. Put the cereals in a bowl
2. Sprinkle some cocoa on the cereals
3. Finally add the milk. Yummy!

Links are very important and formatted in a Wiki-like syntax:

Visit [Wordpress](http://www.wordpress.com)

In case you just want the URL to be displayed as link text:

<http://www.wordpress.com>

Images can be included using their URL. This is an example for the Wordpress logo:

![Wordpress](http://s.wordpress.org/about/images/logos/wordpress-logo-32-blue.png)

Horizontal rules are added by placing minimum three hyphens, asterisks or underscores on a line:

***

Code blocks are written just as usual paragraphs but with an intendation of minimum four spaces or one tab:

    public class getMarkdown() {}

For a single line of inline code just use backticks `.

To emphasize text use either asterisks and/or underscores:

*important*

For the complete syntax, please refer to Daring Fireball.
